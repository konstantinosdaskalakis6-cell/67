DASKALAKIS – Interactive Webpage (HTML/CSS/JavaScript)

This project is a simple interactive webpage built using HTML, CSS, and basic JavaScript. The page demonstrates fundamental front-end web development concepts such as layout design, styling, user interaction, and dynamic page customization.

The webpage presents a professional layout associated with Piraeus Bank, including a header, main content section, image display, interactive buttons, and a fixed footer.

Features

Clean and responsive HTML page structure

Custom CSS styling with a modern layout

Interactive JavaScript buttons

Background color customization

Text color customization

Reset option to restore default page styles

External link that opens the official Piraeus Bank website

Simple alert message interaction

Image integration using a favicon and embedded image

Technologies Used

HTML5 – Page structure

CSS3 – Layout, styling, and responsive design

JavaScript – Interactive features and DOM manipulation

Purpose of the Project

This project is intended as a learning example for beginner web development, showing how to:

Build a structured webpage

Style elements using CSS

Add interactivity using JavaScript

Modify page elements dynamically using button controls

How to Run

Download or clone the repository.

Make sure the image file TRAPEZA.jpg is in the same folder as the HTML file.

Open the .html file in any web browser.
